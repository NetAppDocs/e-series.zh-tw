= 安裝硬體 - EF50 和 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
準備好安裝 EF50 或 EF80 儲存系統後，將機架安裝硬體安裝到機櫃或電信機架中，然後安裝並固定儲存系統。

如果您的儲存系統裝在機櫃中，您可以跳過此程序並前往link:install-cable.html["連接硬體"]。

.開始之前
* 準備一個無靜電的平坦工作區。
* 取得一個防靜電腕帶、並採取防靜態預防措施。
* 請確保您已收到機架安裝硬體包裝內的安裝說明。如果沒有、請參閱 link:../rackmount-hardware.html["機架安裝硬體"^]。
* 請注意與您的儲存系統重量相關的搬運注意事項。參見 link:install-prepare.html#step-2-understand-the-lifting-precautions["了解搬運注意事項"^]。
* 請注意，儲存系統的氣流是從安裝磁碟機的前部進入，從連接埠所在的後部排出。


.步驟
. 依照機架安裝硬體隨附的說明、安裝儲存系統的機架安裝硬體。
+

CAUTION: 將機架安裝硬體安裝在機櫃或機架的最低可用空間中，以防止儲存系統在安裝時傾倒。

. 將儲存系統安裝並固定在機櫃或電信機架中：
+
.. 將儲存系統放置在軌道上，然後從底部支撐儲存系統，將其滑入到位。
.. 使用隨附的安裝螺絲將儲存系統固定到機櫃或電信機架上。


. 將邊框連接至儲存系統的前面。
+
.. 將邊框放置在儲存系統前面，使兩端的孔與儲存系統上的緊固件對齊。
.. 將擋板卡入定位。




.接下來呢？
安裝好儲存系統的硬體後， link:install-cable.html["連接硬體纜線"]。
