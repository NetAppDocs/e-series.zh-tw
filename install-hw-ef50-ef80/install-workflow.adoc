= 安裝與設定 - EF50 和 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要安裝及設定 EF50 或 EF80 儲存系統，您必須準備站台、安裝硬體元件並進行佈線、開啟儲存系統電源，以及完成儲存系統設定與組態。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:install-prepare.html["準備安裝"]
[role="quick-margin-para"]
為了做好安裝準備，請準備好場地，了解儲存系統的吊掛注意事項，打開包裝箱並將內容物與裝箱單進行核對，並註冊儲存系統以獲得支援福利。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:install-hardware.html["安裝硬體"]
[role="quick-margin-para"]
若要安裝硬體、請將機架安裝硬體安裝至機櫃或機架中、然後安裝並固定儲存系統。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:install-cable.html["連接硬體"]
[role="quick-margin-para"]
要連接硬體，請將控制器彼此連接，以實現控制器間鏡像，從而確保系統完全備援，並用於快取鏡像和 I/O 傳輸，然後將控制器連接到主機網路。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:install-power-hardware.html["啟動儲存系統"]
[role="quick-margin-para"]
若要啟動儲存系統、請將電源線插入每個控制器電源供應器、然後將其連接至不同電路上的電源。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]link:install-complete-setup.html["完成儲存系統設定"]
[role="quick-margin-para"]
若要完成儲存系統設定，請連接並設定控制器上的管理連接埠、設定及管理儲存系統，然後驗證主機 I/O 模組傳輸協定是否正確。
