= 在 E 系列 - Linux （ iSCSI ）中設定 multipath.conf 檔案
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
多重路徑.conf檔案是多重路徑精靈的組態檔、即多重路徑d。

multipath.conf檔案會覆寫多路徑的內建組態表。


NOTE: 對於更新的作業系統8.30、NetApp建議使用提供的預設設定。SANtricity

不需要變更/etc/multipath.conf。
