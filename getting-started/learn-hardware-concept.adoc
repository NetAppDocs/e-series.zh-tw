= E系列硬體總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
E系列儲存陣列有多種組態和機型可供選擇。

儲存陣列包括磁碟櫃、控制器、磁碟機、軟體和韌體。陣列可安裝在機架或機櫃中、並可在12、24或60個磁碟機的機櫃中、針對一或兩個控制器提供可自訂的硬體。您可以將儲存陣列從多種介面類型連接至SAN、並連接至各種主機作業系統。

E系列陣列提供下列機型：

* E2800系列-入門級混合式
* EF280系列-入門級All Flash
* EF300系列：入門級All Flash、All NVMe
* E5700系列-中階混合式
* EF570系列-中階All Flash
* EF600系列-中階All Flash、All NVMe
