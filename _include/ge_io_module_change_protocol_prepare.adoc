.步驟
. 如果您不了解儲存系統中主機 I/O 模組的傳輸協定，請使用 SANtricity System Manager 進行驗證。
+
.. 選擇 *Settings* > *System* 。
.. 在 *Additional Settings* 部分中，選取 *Change host I/O protocol* 以開啟該頁面。
.. 請參閱 *Host I/O Protocol* 欄位中顯示的預設協定。
+

NOTE: 如果要查看主機 I/O 模組的可用傳輸協定清單，請在 *Host I/O Protocol* 欄位中選取下拉式功能表。協定清單是根據目前安裝的主機 I/O 模組而定。



. 下一步取決於預設傳輸協定是否為您想要的傳輸協定：
+
[cols="1,2"]
|===
| 如果... | 然後 ... 


 a| 
預設協定就是您想要的協定
 a| 
無需任何操作。您可以退出此程序。



 a| 
預設協定並非您想要的協定
 a| 
繼續執行此程序以變更傳輸協定。

|===
. 為此程序安排停機維護時段。
