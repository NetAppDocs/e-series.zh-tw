機櫃處理x4

4U機架空間：標準19吋（48.30公分）機架、適合下列尺寸的4U機櫃。

*深度*：38.25英英寸（97.16公分）

*寬度*：17.66英吋（44.86公分）

*高度*：6.87吋（17.46公分）

*最大重量*：250磅（113公斤）
