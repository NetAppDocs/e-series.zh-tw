= 主機介面卡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
主機介面卡（HIC）必須安裝在控制器容器內。EF600控制器包含選用HIC上的主機連接埠。HIC內建的主機連接埠稱為HIC連接埠。
