= EF300或EF600電源供應器更換需求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在更換EF300或EF600陣列中的電源供應器之前、請先檢閱下列需求。

* 您必須擁有控制器櫃或磁碟機櫃模型所支援的替換電源供應器。
* 您必須擁有一個ESD腕帶、或已採取其他防靜電預防措施。



NOTE: 請勿混用不同電壓類型的PSU。永遠像這樣更換。
